# ISM Manufacturing PMI Industry Heatmap + Historical Tracker

Live Streamlit app showing the latest ISM Manufacturing PMI with ranked industry growth/contraction heatmap + historical score trends.

**Deployed on Streamlit Cloud** → one-click from GitHub.

Features:
- Auto-scrapes latest PR Newswire release (most reliable source)
- Exact scoring you asked for (+13 → +1 growing, -3 → -1 contracting, 0 unchanged)
- Beautiful colored heatmap (green → red)
- Historical line charts & monthly comparison heatmap (see color changes over time!)
